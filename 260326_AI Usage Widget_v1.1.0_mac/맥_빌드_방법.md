# 맥에서 사용하는 방법

이 폴더는 맥용 빌드 소스입니다.

맥에서 `build-source` 폴더를 연 뒤 터미널에서 아래 명령어를 실행하면 됩니다.

```bash
cd build-source
npm install
npm run dist:mac
```

빌드가 끝나면 아래 경로에 맥용 결과물이 생성됩니다.

```text
build-source/dist/mac
```

그 안에 생성된 파일을 실행해서 사용하면 됩니다.

## 참고

- 서명되지 않은 앱은 처음 실행 시 macOS 보안 경고가 나올 수 있습니다.
- 이런 경우 앱 아이콘 우클릭 후 `열기`를 선택하거나, 시스템 설정에서 실행을 허용하면 됩니다.
